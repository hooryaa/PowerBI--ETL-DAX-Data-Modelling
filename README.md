# Superstore Sales Analytics Dashboard (Power BI)

This project is an end-to-end **data analytics solution** built using Power BI on the popular Superstore dataset from Kaggle. It focuses on transforming raw sales data into meaningful business insights through **ETL, data modeling, DAX calculations, and interactive visualizations**.

---

## 📊 Project Overview

The goal of this project is to analyze retail sales performance and uncover key business insights such as:

- Sales and profit trends
- Top-performing products and categories
- Regional performance comparison
- Customer segmentation insights
- Shipping and order efficiency analysis

---

## 🧰 Workflow

### 1. ETL (Extract, Transform, Load)
- Imported raw Superstore dataset from Kaggle
- Cleaned missing and inconsistent values
- Removed duplicates and unnecessary columns
- Transformed data types for proper analysis
- Structured data for reporting

---

### 2. Data Modeling
- Built a **star schema model**
- Created relationships between fact and dimension tables
- Optimized model for performance and scalability
- Ensured proper filtering flow across visuals

---

### 3. DAX (Data Analysis Expressions)
Implemented key measures such as:

- Total Sales
- Total Profit
- Profit Margin
- Sales Growth
- Year-to-Date (YTD) metrics
- Category & Region-wise KPIs

---

### 4. Visualization & Dashboard
Designed an **interactive Power BI dashboard** featuring:

- KPI cards (Sales, Profit, Orders)
- Time-series analysis of sales trends
- Category and sub-category breakdown
- Regional performance heatmaps
- Customer segmentation insights
- Dynamic filters and slicers for exploration

---

## 📈 Key Insights

- Identified top-performing product categories driving revenue
- Highlighted regions with negative profit margins
- Discovered seasonal trends in sales performance
- Found high-revenue but low-profit product segments
- Analyzed shipping delays affecting customer satisfaction
